# Sentiment and Hashtag-aware Attentive Deep Neural Network for Multimodal Post Popularity Prediction

## Quick Facts
- arXiv ID: 2412.10737
- Source URL: https://arxiv.org/abs/2412.10737
- Reference count: 40
- Primary result: NARRATOR achieves 62.625% and 67.709% improvements in MSE, and 40.487% and 50.051% in MAE on TPIC and SMP datasets respectively

## Executive Summary
This paper introduces NARRATOR, a deep learning framework for predicting social media post popularity by integrating multimodal content features, hashtag semantics, visual demographics, and sentiment analysis. The proposed hashtag-guided attention mechanism leverages hashtag embeddings to dynamically focus on relevant text and image features, while visual demographic information is extracted from faces in images to capture audience engagement trends. Sentiment analysis of hashtags provides additional insights into emotional resonance. Experiments on real-world datasets show NARRATOR significantly outperforms state-of-the-art methods.

## Method Summary
NARRATOR is a multimodal deep learning framework that combines textual content, visual features, hashtag semantics, visual demographics, and sentiment analysis for social media post popularity prediction. The framework employs a hashtag-guided attention mechanism that uses hashtag embeddings to dynamically weight the importance of text and image features. Visual demographic features are extracted from faces in images using demographic models, while sentiment analysis captures emotional tones of hashtags. The model integrates these multimodal signals through a unified architecture to predict post popularity with improved accuracy.

## Key Results
- Achieved 62.625% improvement in MSE on TPIC dataset
- Achieved 67.709% improvement in MSE on SMP dataset
- Achieved 40.487% and 50.051% improvements in MAE on TPIC and SMP datasets respectively

## Why This Works (Mechanism)
NARRATOR works by effectively integrating multimodal signals through specialized attention mechanisms. The hashtag-guided attention leverages hashtag embeddings to dynamically focus on relevant text and image features, allowing the model to identify which aspects of the content are most likely to drive engagement. Visual demographic extraction captures audience characteristics from images, while hashtag sentiment analysis provides emotional context. These components work synergistically, with the attention mechanism serving as the central coordinating element that weights different modalities based on their relevance to popularity prediction.

## Architecture Onboarding

**Component Map:**
Text features -> Attention layer -> Popularity prediction
Image features -> Attention layer -> Popularity prediction
Hashtag embeddings -> Attention layer -> Popularity prediction
Visual demographics -> Feature fusion -> Popularity prediction
Hashtag sentiment -> Feature fusion -> Popularity prediction

**Critical Path:**
Hashtag embeddings -> Hashtag-guided attention -> Weighted feature fusion -> Prediction layer

**Design Tradeoffs:**
- Complexity vs. performance: NARRATOR's sophisticated multimodal integration yields significant improvements but increases computational requirements
- Dependency on hashtags: The attention mechanism relies heavily on hashtag availability and quality, potentially limiting applicability to platforms with fewer or less semantic hashtags
- Visual demographic extraction: Requires detectable faces in images, which may not always be present or may be challenging to analyze accurately

**Failure Signatures:**
- Poor performance on posts without hashtags or with semantically weak hashtags
- Reduced accuracy when images lack detectable faces or contain ambiguous facial features
- Degradation in performance when hashtag sentiment classification is noisy or inaccurate

**First 3 Experiments:**
1. Ablation study removing hashtag-guided attention to measure its contribution
2. Testing with posts that have no hashtags to evaluate baseline performance
3. Cross-platform evaluation on different social media datasets to assess generalizability

## Limitations
- Evaluation limited to two specific datasets (TPIC and SMP), constraining generalizability to other platforms
- Reliance on hashtag embeddings assumes sufficient semantic richness in hashtags, which may not hold for all social media contexts
- Visual demographic extraction depends on face detection accuracy and may not generalize well to images without faces
