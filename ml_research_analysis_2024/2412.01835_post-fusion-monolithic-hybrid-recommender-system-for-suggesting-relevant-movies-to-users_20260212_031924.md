# Post-fusion monolithic hybrid recommender system for suggesting relevant movies to users
## Quick Facts
- arXiv ID: 2412.01835
- Source URL: https://arxiv.org/abs/2412.01835
- Reference count: 37
- Proposes hybrid post-fusion monolithic recommender combining sequential analysis and ratings-based collaborative filtering

## Executive Summary
This study introduces a hybrid post-fusion monolithic recommender system that dynamically combines two collaborative filtering approaches to enhance movie recommendations. The system leverages sequential analysis of watched movies alongside traditional rating-based predictions, adaptively weighting each source based on data availability. The approach aims to capture both short-term and long-term user preferences through self-attention mechanisms for sequential modeling and deep neural networks for rating prediction.

## Method Summary
The proposed method employs a hybrid architecture that combines sequential analysis of watched movies with traditional rating-based collaborative filtering. The system uses self-attention mechanisms to model sequential patterns in user behavior and deep neural networks to predict ratings. A key innovation is the dynamic weighting mechanism that assigns higher importance to ratings when available and to sequential data when ratings are sparse. The post-fusion approach integrates these two sources at the recommendation level rather than at the feature level, allowing for more flexible adaptation to different user behaviors and data availability scenarios.

## Key Results
- The hybrid approach outperforms individual models in terms of NDCG and RMSE metrics on the MovieLens dataset
- Dynamic weighting effectively balances sequential and rating-based recommendations based on data availability
- The system successfully captures both short-term and long-term user preferences through the combination of self-attention and deep neural networks

## Why This Works (Mechanism)
The system works by combining two complementary sources of user preference information: sequential patterns from watched movies and explicit ratings. The self-attention mechanism captures temporal dependencies and short-term interests from the sequence of watched movies, while the deep neural network models long-term preferences through rating patterns. The dynamic weighting allows the system to adapt to different user scenarios - when users have provided explicit ratings, those carry more weight, but when ratings are sparse, the sequential patterns become more influential. This post-fusion approach allows for more flexible integration than traditional feature-level fusion methods.

## Architecture Onboarding
**Component map**: User history -> Sequential model (self-attention) -> Rating model (deep neural network) -> Dynamic weighting -> Final recommendations
**Critical path**: User behavior data flows through both sequential and rating models in parallel, then their outputs are weighted and combined to generate final recommendations
**Design tradeoffs**: Post-fusion approach provides flexibility but may miss feature-level interactions; dynamic weighting adds complexity but improves adaptation to data availability
**Failure signatures**: Poor performance when user histories are too short for effective sequential modeling; degradation when rating data is sparse and sequential patterns are weak
**First experiments**: 1) Baseline evaluation with individual models only, 2) Test dynamic weighting effectiveness across different user segments, 3) Validate self-attention attention weights on sequential patterns
