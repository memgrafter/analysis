# An attempt to generate new bridge types from latent space of denoising diffusion Implicit model

## Quick Facts
- arXiv ID: 2402.07129
- Source URL: https://arxiv.org/abs/2402.07129
- Authors: Hongjun Zhang
- Reference count: 9
- Generates five new asymmetric bridge types using DDIM trained on symmetric three-span bridges

## Executive Summary
This paper presents a method for generating new bridge types using denoising diffusion implicit models (DDIM). The approach trains on a dataset of 9,600 grayscale images of symmetric three-span bridges (beam, arch, cable-stayed, and suspension) and generates novel asymmetric designs by sampling from the latent space. The model successfully produced five new bridge types with feasible structural technologies, demonstrating DDIM's capability for creative bridge-type innovation.

## Method Summary
The study employs a DDIM model based on TensorFlow and Keras, using a U-Net architecture to learn the noise-adding process from a dataset of symmetric three-span bridges. The model was trained on 192x48 pixel grayscale images and generates new asymmetric bridge designs by sampling from the latent space and iteratively denoising. The implementation includes sinusoidal embedding of noise variance, ResidualBlocks, DownBlocks, and UpBlocks, with deterministic generation achieved by setting σₜ = 0 during the denoising process.

## Key Results
- Model trained on 9,600 symmetric bridge images achieved stable training with decreasing loss curve
- Five new asymmetric bridge types successfully generated with feasible structural technologies
- DDIM demonstrated creative capability in bridge-type innovation through latent space sampling

## Why This Works (Mechanism)

### Mechanism 1
- Claim: DDIM enables asymmetric bridge generation by reversing the noise-adding process learned during training.
- Mechanism: The model learns to predict noise added at each step; starting from random latent noise, it iteratively denoises to generate images. By sampling from latent space, the model can combine structural components in ways not seen in the training set.
- Core assumption: The learned reverse process is smooth enough to produce meaningful structural combinations not explicitly present in training data.
- Evidence anchors:
  - [abstract] "From the latent space sampling, new bridge types with asymmetric structures can be generated."
  - [section] "From the latent space sampling, new bridge types with asymmetric structures can be generated."
- Break condition: If the latent space does not capture sufficient structural diversity, or if the reverse process is unstable, generated bridges may be implausible.

### Mechanism 2
- Claim: The dataset of symmetric bridges allows the model to learn generalizable structural components.
- Mechanism: By training on symmetric three-span bridges (beam, arch, cable-stayed, suspension), the model learns to represent structural elements that can be recombined asymmetrically during generation.
- Core assumption: Symmetric bridge components are sufficient to reconstruct or invent new asymmetric configurations.
- Evidence anchors:
  - [section] "Using symmetric structured image dataset of three-span beam bridge, arch bridge, cable-stayed bridge and suspension bridge."
- Break condition: If the symmetric dataset lacks sufficient structural variety, the model cannot generate diverse asymmetric types.

### Mechanism 3
- Claim: DDIM's latent space sampling produces distinct outputs for each point, enabling exploration of new bridge types.
- Mechanism: By setting σₜ = 0 during denoising, the model enforces deterministic mapping from latent space to pixel space, so each sampled latent point yields a unique bridge design.
- Evidence anchors:

- Break condition: If σₜ were nonzero, outputs would be too varied to reliably explore structured design space.

## Architecture Onboarding

- Component map: Image preprocessing (normalization) -> DDIM noise addition process (q function) -> U-Net neural network (predicts noise from noisy images) -> Latent space sampling -> Iterative denoising to generate images
- Critical path: Latent space sampling → U-Net noise prediction → Iterative denoising → Final bridge image
- Design tradeoffs:
  - Dataset resolution reduced to 192x48 for faster training but at cost of image clarity.
  - Deterministic sampling (σₜ = 0) sacrifices diversity for reproducibility.
  - Symmetric training data enables asymmetric generation but may limit structural variety.
- Failure signatures:
  - Loss curve not decreasing → training instability or poor architecture choice.
  - Generated images unrealistic → noise prediction inaccurate or insufficient training data.
  - Outputs too similar → latent space not diverse or deterministic sampling too restrictive.
- First 3 experiments:
  1. Train on small subset of symmetric bridges, check loss curve stability.
  2. Sample latent space, generate a few outputs, inspect for structural plausibility.
  3. Gradually increase dataset size, observe improvement in generated bridge diversity and realism.
