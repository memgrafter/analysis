# Modeling Time-Variant Responses of Optical Compressors with Selective State Space Models

## Quick Facts
- arXiv ID: 2408.12549
- Source URL: https://arxiv.org/abs/2408.12549
- Reference count: 7
- Key outcome: S6-based neural network achieves superior emulation of optical compressors compared to LSTM, Encoder-Decoder, and S4D models

## Executive Summary
This paper introduces a method for modeling optical dynamic range compressors using Selective State Space (S6) models. The approach combines S6 blocks for input encoding with Feature-wise Linear Modulation and Gated Linear Units to dynamically adjust network parameters based on external controls. The architecture is specifically designed for low-latency, real-time applications suitable for live audio processing. Validation demonstrates superior performance compared to state-of-the-art models across multiple metrics when applied to analog optical compressors.

## Method Summary
The proposed method employs S6 blocks to encode input audio signals, which are then processed through Feature-wise Linear Modulation layers that adjust the network behavior based on external parameters. Gated Linear Units are incorporated to control information flow and enhance the model's ability to capture time-variant responses. The architecture is designed with real-time constraints in mind, prioritizing low-latency operation for live audio applications. The model is trained and validated on two analog optical compressors (TubeTech CL 1B and Teletronix LA-2A) across various control parameter settings.

## Key Results
- Superior performance compared to LSTM, Encoder-Decoder, and S4D models on optical compressor emulation
- Accurate modeling for both seen and unseen parameter settings
- Notable correlation between accuracy and control parameter sampling density
- Fast attack with slow release identified as most challenging parameter combination to emulate

## Why This Works (Mechanism)
The S6 model's ability to handle long-range dependencies and selective state transitions makes it particularly effective for capturing the complex, time-varying behavior of optical compressors. The integration of Feature-wise Linear Modulation allows the network to dynamically adapt its internal representations based on external control parameters, which is crucial for modeling the nonlinear characteristics of analog compressors. Gated Linear Units provide additional control over information flow, helping the model learn when to update its state based on the input signal characteristics.

## Architecture Onboarding

**Component Map:**
Input Audio -> S6 Encoder -> FiLM Layers -> GLU -> Output

**Critical Path:**
The critical path runs from input audio through the S6 encoder, where temporal dependencies are captured, then through FiLM layers that modulate the features based on external parameters, followed by GLU for controlled information flow, and finally to the output prediction.

**Design Tradeoffs:**
The architecture prioritizes real-time performance over maximum accuracy, using S6 models which offer better computational efficiency than Transformers for long sequences. The use of FiLM layers introduces parameter conditioning but adds computational overhead. GLUs provide better gradient flow but require additional parameters.

**Failure Signatures:**
The model shows reduced accuracy for fast attack with slow release settings, indicating potential limitations in capturing extreme temporal dynamics. Lower sampling density of control parameters leads to decreased accuracy, suggesting the model may overfit to training distributions.
