# Knowledge Discovery using Unsupervised Cognition

## Quick Facts
- arXiv ID: 2409.20064

## Executive Summary
This paper presents a knowledge discovery pipeline using the Unsupervised Cognition algorithm that combines pattern mining, correlation-based feature selection, and dimensionality reduction. The method extracts patterns from model representations, selects features based on their correlation with target variables, and reduces dimensionality by retaining only relevant features. When applied to the TCGA Kidney Cancers dataset with 58,056 features and 897 samples, the approach achieves dramatic dimensionality reduction (96.44% to 99.495%) while improving classification accuracy from 83.69% to 89.98%, outperforming traditional methods like PCA and SOM.

## Method Summary
The pipeline operates through three main stages: pattern mining using Unsupervised Cognition to extract representations from high-dimensional data, correlation analysis to identify features most strongly associated with the target variable, and dimensionality reduction by retaining only these selected features. The approach is validated on the TCGA Kidney Cancers dataset, demonstrating significant dimensionality reduction while maintaining or improving classification performance compared to baseline methods.

## Key Results
- Reduced dimensionality by 96.44% (58,056 to 2,067 features) with accuracy improvement from 83.69% to 87.59%
- Further feature selection reduced dimensions by 99.495% (to 291 features) with accuracy improving to 89.98%
- Outperformed PCA and SOM in both dimensionality reduction and accuracy gains on kidney cancer dataset

## Why This Works (Mechanism)
The pipeline works by leveraging pattern mining to discover meaningful representations in high-dimensional data, then using correlation analysis to identify the most informative features for the prediction task. This targeted approach avoids the information loss inherent in traditional dimensionality reduction methods like PCA, which may discard features that individually appear weak but collectively contain important signals. By focusing on features with strong relationships to the target variable, the method maintains predictive power while dramatically reducing computational complexity.

## Architecture Onboarding

Component map: High-dimensional data -> Unsupervised Cognition pattern mining -> Correlation analysis with target -> Feature selection -> Reduced dimensionality data

Critical path: The pipeline's critical path is the sequential flow from pattern mining through correlation analysis to final feature selection, as each stage depends on the output of the previous one

Design tradeoffs: The method trades computational simplicity for interpretability, as correlation-based selection provides clearer feature-to-target relationships than black-box methods like PCA, but may miss complex interactions between features

Failure signatures: Performance degradation would likely manifest as either insufficient dimensionality reduction (failing to reduce computational burden) or accuracy loss (discarding important features)

First experiments: 1) Run correlation analysis on raw data without pattern mining to isolate its contribution, 2) Compare selected features against domain knowledge for biological relevance, 3) Test sensitivity to correlation threshold parameters
