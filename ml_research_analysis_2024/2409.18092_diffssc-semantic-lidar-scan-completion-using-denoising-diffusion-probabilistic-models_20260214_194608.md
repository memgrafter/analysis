# DiffSSC: Semantic LiDAR Scan Completion using Denoising Diffusion Probabilistic Models

## Quick Facts
- **arXiv ID**: 2409.18092
- **Source URL**: https://arxiv.org/abs/2409.18092
- **Reference count**: 39
- **Primary result**: Achieves state-of-the-art semantic LiDAR scan completion with IoU(SC) of 49.38 and mIoU(SSC) of 22.67 on SemanticKITTI

## Executive Summary
DiffSSC introduces a novel approach for semantic LiDAR scan completion that addresses the challenges of sparse and semantically incomplete 3D point clouds in autonomous driving. The method leverages denoising diffusion probabilistic models (DDPMs) to jointly predict missing geometry and semantics, extending DDPMs to operate directly on point clouds without voxelization. By separating point and semantic space modeling through forward diffusion and reverse denoising processes, DiffSSC achieves superior performance on benchmark datasets while avoiding quantization errors inherent in voxel-based approaches.

## Method Summary
The DiffSSC framework combines a two-stage approach with denoising diffusion probabilistic models to complete semantic LiDAR scans. First, a Cylinder3D model provides initial semantic segmentation of the sparse input point cloud. This segmentation serves as conditional input for the diffusion model, which then generates complete geometric and semantic information through separate forward diffusion and reverse denoising processes for point and semantic spaces. Local and global regularization losses are employed to stabilize training and improve generation quality. The approach operates directly on raw point clouds, avoiding the quantization errors that arise from voxelization while maintaining efficient processing capabilities.

## Key Results
- Achieves IoU(SC) of 49.38 on SemanticKITTI benchmark
- Achieves mIoU(SSC) of 22.67 on SemanticKITTI benchmark
- Outperforms existing state-of-the-art approaches on both SemanticKITTI and SSCBench-KITTI360 datasets

## Why This Works (Mechanism)
The diffusion-based approach works by modeling the uncertainty and complexity of LiDAR data through gradual noise addition (forward diffusion) and subsequent denoising (reverse process). This probabilistic framework naturally handles the sparsity and incompleteness of LiDAR scans by learning the underlying data distribution. The separation of point and semantic space modeling allows the network to focus on geometric completion and semantic refinement independently, while the conditional input from Cylinder3D segmentation guides the generation process toward semantically coherent results.

## Architecture Onboarding

**Component Map**
Raw Point Cloud -> Cylinder3D Segmentation -> Diffusion Model (Point Space + Semantic Space) -> Completed Semantic Point Cloud

**Critical Path**
Input point cloud → Cylinder3D semantic segmentation → Conditional diffusion generation → Final semantic scene completion

**Design Tradeoffs**
- Raw point cloud processing vs voxel-based methods: Avoids quantization but may have higher computational complexity
- Separate point and semantic space modeling vs joint modeling: Allows specialized treatment but requires careful coordination
- Two-stage approach vs end-to-end: Provides strong semantic priors but introduces dependency on initial segmentation quality

**Failure Signatures**
- Poor initial segmentation from Cylinder3D leading to semantically inconsistent completions
- Diffusion model collapse when dealing with highly sparse or occluded regions
- Generation artifacts at boundaries between different semantic classes
