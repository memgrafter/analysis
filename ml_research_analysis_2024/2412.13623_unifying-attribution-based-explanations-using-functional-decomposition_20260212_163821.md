# Unifying Attribution-Based Explanations Using Functional Decomposition

## Quick Facts
- arXiv ID: 2412.13623
- Source URL: https://arxiv.org/abs/2412.13623
- Authors: Arne Gevaert; Yvan Saeys
- Reference count: 39
- Key outcome: This paper presents a unifying framework for attribution-based explanation methods in machine learning, addressing the challenge of method selection and evaluation.

## Executive Summary
This paper presents a unifying framework for attribution-based explanation methods in machine learning, addressing the challenge of method selection and evaluation. The core idea is to define removal-based attribution methods (RBAMs) and link them to additive functional decomposition through the canonical additive decomposition (CAD). The authors show that many existing methods, including Shapley value-based, permutation-based, and variance-based explanations, can be viewed as RBAMs. They introduce a formal connection between RBAMs and cooperative game theory, demonstrating that any RBAM can be defined as a game-theoretic value or interaction index. The paper derives sufficient conditions for RBAMs to adhere to functional axioms, providing a rigorous way to study their behavior. Finally, the authors show how this framework can be used to develop efficient approximations for existing explanation methods, potentially leading to faster algorithms and better understanding of model behavior.

## Method Summary
The paper introduces a unifying framework for attribution-based explanation methods by defining removal-based attribution methods (RBAMs) as linear combinations of model behaviors after feature removal. The framework links RBAMs to additive functional decomposition through the canonical additive decomposition (CAD) and establishes a formal connection to cooperative game theory. The authors derive sufficient conditions for RBAMs to adhere to functional axioms and demonstrate how this framework can be used to develop efficient approximations for existing explanation methods. The method involves three key mathematical choices: behavior mapping, feature removal operators, and aggregation coefficients, which together define any RBAM.
