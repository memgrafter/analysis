# MIML library: a Modular and Flexible Library for Multi-instance Multi-label Learning

## Quick Facts
- **arXiv ID**: 2402.08056
- **Source URL**: https://arxiv.org/abs/2402.08056
- **Reference count**: 40
- **Primary result**: Java library with 43 algorithms for MIML learning, extensible via XML configuration without programming

## Executive Summary
MIML library is a Java software tool designed to develop, test, and compare classification algorithms for multi-instance multi-label (MIML) learning. The library extends the Weka and Mulan architectures to create a unified framework that handles both multi-instance and multi-label aspects simultaneously. It provides comprehensive facilities for data management, evaluation through holdout and cross-validation methods, standard performance metrics, and XML-based configuration for algorithm execution without programming knowledge.

## Method Summary
The library implements 43 classification algorithms for MIML problems, organized into transformations to multi-instance/multi-label, neural networks, optimization-based, and instance-based methods. Users can execute algorithms through XML configuration files specifying classifiers, evaluators, and reports. The library supports cross-validation and holdout experiments, generates performance metrics including Hamming Loss and Precision/Recall measures, and produces reports in various formats. MIML library is built on Weka and Mulan frameworks, making it platform-independent, extensible, and available under GNU GPL license.
