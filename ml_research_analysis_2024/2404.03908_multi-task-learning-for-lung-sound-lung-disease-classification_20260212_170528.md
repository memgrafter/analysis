# Multi-Task Learning for Lung sound & Lung disease classification

## Quick Facts
- arXiv ID: 2404.03908
- Source URL: https://arxiv.org/abs/2404.03908
- Authors: Suma K; Deepali Koppad; Preethi Kumar; Neha A Kantikar; Surabhi Ramesh
- Reference count: 40
- Primary result: Multi-task learning framework achieves 74% lung sound classification accuracy and 91% lung disease classification accuracy

## Executive Summary
This paper presents a multi-task learning (MTL) framework for simultaneous classification of lung sounds and lung diseases using deep learning models. The approach leverages shared feature representations to improve classification accuracy while reducing computational redundancy. Using the ICBHI 2017 Respiratory Sound Database, the MTL approach with MobileNet architecture achieved the highest accuracy of 74% for lung sound classification and 91% for lung disease classification. Additionally, the study explored risk level computation for COPD using demographic data and three machine learning algorithms, with Random Forest demonstrating 92% accuracy for risk prediction.

## Method Summary
The study employed multi-task learning using four deep learning architectures (2D CNN, ResNet50, MobileNet, and DenseNet) to classify lung sounds and lung diseases simultaneously. MFCC features were extracted from lung sound recordings and used as input to the shared feature extraction backbone. MobileNet's depthwise separable convolutions were particularly effective in reducing computational complexity while maintaining accuracy. For risk level prediction, demographic data (age, gender, BMI) was used with three classifiers (Logistic Regression, SVM, and Random Forest) to predict COPD risk levels. The ICBHI 2017 Respiratory Sound Database served as the primary dataset, containing 920 audio recordings from 126 patients.

## Key Results
- MTL approach with MobileNet achieved 74% accuracy for lung sound classification and 91% for lung disease classification
- Random Forest classifier demonstrated 92% accuracy for COPD risk level prediction using demographic data
- MobileNet architecture showed superior performance among the four deep learning models tested
- Risk level computation based on age, gender, and BMI provided clinically relevant severity assessment

- Claim: Multi-task learning improves classification accuracy by sharing lower-level features between lung sound and disease classification tasks.
- Mechanism: Shared feature extractor learns common representations from MFCC features, reducing redundancy and enabling complementary task learning.
- Core assumption: Lung sound and disease classification tasks are related enough to benefit from shared representations.

### Mechanism 2
- Claim: MobileNet architecture's depthwise separable convolutions reduce computational complexity while maintaining classification accuracy.
- Mechanism: Depthwise convolutions process each input channel separately, followed by pointwise convolutions to combine features, reducing parameters and computations.
- Core assumption: Computational efficiency can be maintained without sacrificing accuracy for lung sound and disease classification.
- Evidence anchors:
  - [section] "The core of MobileNet architecture is the idea of depthwise separable convolutions, which divide normal convolutions into two different processes: pointwise and depthwise convolution."
  - [section] "MobileNet has demonstrated the highest accuracy for both sound and disease categorization among the four deep learning models."
  - [corpus] Weak evidence - no corpus papers directly compare MobileNet performance for lung sound classification.

## Architecture Onboarding

- Component map:
  MFCC feature matrix -> Shared MobileNet backbone -> Lung sound classification head -> Disease classification head
  Demographic data -> Random Forest/SVM/Logistic Regression -> COPD risk prediction

- Critical path:
  MFCC feature extraction → MobileNet feature learning → Joint classification heads → Risk prediction pipeline

- Design tradeoffs:
  - MobileNet vs ResNet50: MobileNet offers better computational efficiency but may sacrifice some accuracy
  - Shared vs separate feature extractors: Shared features improve efficiency but may limit task-specific optimization
  - MTL vs single-task learning: MTL reduces training time but may introduce conflicting gradients

- Failure signatures:
  - Low accuracy in both tasks: Feature sharing may be too aggressive, preventing task-specific optimization
  - High accuracy in one task, low in another: Imbalance in task difficulty or data distribution
  - Risk prediction performs poorly: Demographic factors may not be strong predictors for this population

## Limitations
- Small dataset size (126 patients) limits generalizability across diverse populations
- Lack of detailed information about data preprocessing parameters and cross-validation procedures
- No ablation studies to isolate specific contributions of MTL architecture versus individual task performance
- Risk prediction relies solely on demographic factors without incorporating clinical biomarkers

## Confidence
- Multi-task learning effectiveness: Medium - Results show improved performance but lack comparative analysis against single-task baselines
- MobileNet superiority: Medium - Only relative performance among four architectures is reported, without external validation
- Demographic-based risk prediction: Low - Limited to basic patient characteristics without clinical validation or comparison to established risk assessment tools
