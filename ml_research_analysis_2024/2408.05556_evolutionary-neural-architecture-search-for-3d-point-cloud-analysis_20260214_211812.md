# Evolutionary Neural Architecture Search for 3D Point Cloud Analysis

## Quick Facts
- arXiv ID: 2408.05556
- Source URL: https://arxiv.org/abs/2408.05556
- Authors: Yisheng Yang; Guodong Du; Chean Khim Toa; Ho-Kin Tang; Sim Kuan Goh
- Reference count: 24
- Primary result: SHSADE-PIDS achieves 64.51% mIoU on SemanticKITTI with 0.55M parameters and 4.5GMACs

## Executive Summary
This paper introduces SHSADE-PIDS, an evolutionary neural architecture search method that bridges discrete and continuous spaces to optimize neural networks for 3D point cloud tasks. The approach combines the continuous optimization strengths of SHSADE with discrete-continuous mapping to enable efficient exploration of architecture configurations. Comprehensive experiments on semantic segmentation using SemanticKITTI and shape classification using ModelNet40 demonstrate SHSADE-PIDS's capabilities in discovering specialized models for point cloud processing. The neural architectures found by SHSADE-PIDS significantly advance higher accuracy and efficiency over prior works, including both hand-designed networks and other NAS approaches.

## Method Summary
SHSADE-PIDS is an evolutionary neural architecture search method that leverages continuous optimization techniques to search for optimal neural architectures for 3D point cloud analysis. The method bridges discrete and continuous search spaces through a hybrid mapping approach, allowing for more efficient exploration of architecture configurations. By combining evolutionary algorithms with continuous search space optimization, SHSADE-PIDS can discover specialized neural architectures that are both accurate and efficient for point cloud processing tasks. The approach is evaluated on two major tasks: semantic segmentation on SemanticKITTI and shape classification on ModelNet40, demonstrating superior performance compared to both hand-designed networks and other NAS approaches.

## Key Results
- Achieves 64.51% mIoU on SemanticKITTI semantic segmentation with only 0.55M parameters and 4.5GMACs
- Secures 93.4% accuracy on ModelNet40 shape classification with just 1.31M parameters
- Reduces computational overhead by 22-26X compared to top competing methods while maintaining or improving accuracy

## Why This Works (Mechanism)
SHSADE-PIDS works by effectively leveraging continuous evolutionary algorithms for discrete architecture search. The key mechanism involves mapping continuous search parameters to discrete architecture configurations, which allows for more efficient exploration of the architecture space. This hybrid approach combines the benefits of continuous optimization (smooth gradients, efficient search) with the discrete nature of neural architectures. The evolutionary component ensures that promising architectures are retained and refined across generations, while the continuous mapping enables faster convergence to optimal solutions.

**Design Tradeoffs**: The method balances search efficiency against architecture expressiveness. Continuous search enables faster exploration but requires careful mapping to discrete architectures. ## Limitations
- Evaluation primarily focused on SemanticKITTI and ModelNet40 datasets, potentially limiting generalizability to other point cloud applications
- Continuous-to-discrete mapping approach may face scalability challenges when applied to more complex architecture search spaces
- Paper lacks detailed discussion of computational costs during the search phase
- Efficiency claims depend on implementation details and comparison baselines that need careful scrutiny
