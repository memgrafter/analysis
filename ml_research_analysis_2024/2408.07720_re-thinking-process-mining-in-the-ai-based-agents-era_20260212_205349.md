# Re-Thinking Process Mining in the AI-Based Agents Era

## Quick Facts
- arXiv ID: 2408.07720
- Source URL: https://arxiv.org/abs/2408.07720
- Reference count: 23

## Executive Summary
This paper presents a novel approach to process mining by integrating AI-based agents with Large Language Models (LLMs). The proposed AI-Based Agents Workflows (AgWf) paradigm addresses the limitations of direct LLM approaches in complex reasoning tasks by decomposing processes into simpler workflows and combining deterministic tools with LLM domain knowledge. The framework leverages specialized AI-based tasks for better task decomposition, tool selection, and result synthesis. Through implementation using the CrewAI framework and demonstration in two case studies (fairness assessment and root cause analysis), the approach shows improved performance over traditional direct LLM methods in process mining applications.

## Method Summary
The paper introduces a systematic approach to enhance process mining tasks using AI-based agents through the AgWf paradigm. The method decomposes complex process mining tasks into simpler workflows, integrating deterministic tools with LLM domain knowledge. This involves task decomposition, tool selection, and result synthesis through specialized AI-based tasks. The CrewAI framework is presented as an implementation of this approach, demonstrating how multiple agents can collaborate on different aspects of process mining workflows. The method particularly addresses the challenge of complex reasoning in process mining by distributing tasks across specialized agents with domain-specific capabilities.

## Key Results
- Demonstrated improved performance over direct LLM approaches through workflow decomposition and specialized agents
- Successfully applied to fairness assessment and root cause analysis in process mining contexts
- Showed effective integration of deterministic tools with LLM domain knowledge through the CrewAI framework implementation

## Why This Works (Mechanism)
The approach works by decomposing complex process mining tasks into manageable workflows that leverage both deterministic tools and LLM capabilities. By distributing reasoning tasks across specialized AI-based agents, each handling specific aspects of the process mining pipeline, the system can achieve better performance than monolithic LLM approaches. The workflow-based architecture allows for better tool selection and result synthesis, addressing the limitations of LLMs in complex reasoning scenarios through collaborative multi-agent systems.
