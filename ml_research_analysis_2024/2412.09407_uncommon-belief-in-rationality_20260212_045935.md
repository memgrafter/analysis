# Uncommon Belief in Rationality

## Quick Facts
- arXiv ID: 2412.09407
- Source URL: https://arxiv.org/abs/2412.09407
- Reference count: 40
- Primary result: Introduces RBR graphs and doxastic rationalisability solution concept for modeling complex belief structures about agent rationality

## Executive Summary
This paper presents a novel graph-based language for representing higher-order beliefs about agent rationality in multi-agent systems. The framework extends beyond traditional common knowledge assumptions by allowing for uncommon or incomplete beliefs about other agents' rationality. The authors develop a solution concept called doxastic rationalisability and prove that any RBR graph can be compressed into a unique minimal form through a polynomial-time algorithm. This work bridges game theory and epistemic logic by providing a formal structure for modeling strategic reasoning under diverse belief hierarchies.

## Method Summary
The authors define RBR (rationality and beliefs in rationality) graphs as the fundamental representation for belief structures in multi-agent systems. They introduce the doxastic rationalisability solution concept, which captures how agents reason about each other's rationality within these belief hierarchies. The core technical contribution is an efficient algorithm that compresses any RBR graph into a minimal equivalent form, preserving strategic properties while reducing complexity. This minimization process enables more tractable analysis of belief structures while maintaining the essential strategic relationships between agents.

## Key Results
- Proves that any RBR graph can be uniquely minimized while preserving strategic equivalence
- Develops a polynomial-time algorithm for RBR graph compression
- Establishes theoretical foundations for modeling uncommon beliefs about rationality
- Bridges epistemic logic and game theory through graph-based representation

## Why This Works (Mechanism)
The framework works by representing belief hierarchies as directed graphs where nodes capture agents' beliefs about rationality and edges represent belief dependencies. The minimization algorithm exploits structural redundancies in these graphs to produce equivalent but simpler representations. By formalizing the notion of doxastic rationalisability, the method provides a principled way to analyze strategic reasoning under diverse belief structures, allowing agents to have different and potentially incomplete beliefs about each other's rationality.

## Limitations
- Computational tractability for large-scale multi-agent systems remains unproven
- Complexity analysis of the minimization algorithm is incomplete
- Framework assumes perfect rationality, which may not reflect real-world behavior
- Scalability concerns when modeling systems with many agents or deep belief hierarchies
