# Weighted Embeddings for Low-Dimensional Graph Representation

## Quick Facts
- arXiv ID: 2410.06042
- Source URL: https://arxiv.org/abs/2410.06042
- Reference count: 13
- Primary result: WEMBED achieves F1-score of 0.9999 for citeseer network with 8 dimensions in under 11 seconds

## Executive Summary
This paper introduces weighted embeddings (WEMBED) as a mathematically simpler alternative to hyperbolic embeddings for representing heterogeneous graphs. The method maps nodes to positions in Euclidean space with assigned weights, using weighted distances as a similarity measure. WEMBED demonstrates superior performance on heterogeneous graph reconstruction tasks while maintaining competitive running times with state-of-the-art Euclidean embedders.

## Method Summary
WEMBED uses weighted distances dist(u,v) = ||pu-pv||/(wuwv)^(1/d) as a similarity measure, where pu and pv are node positions and wu and wv are weights. The algorithm initializes weights as wv = deg(v)^(d/8) and random positions, then optimizes using batch gradient descent with Adam optimizer. The linear loss function combines positive edges (uv∈E) with max(dist(u,v)-ℓ,0) and negative edges (uv∉E) with max(ℓ-dist(u,v),0). An R-tree data structure partitioned by weight classes efficiently finds non-neighbors within threshold distance for gradient computation.

## Key Results
- WEMBED achieves F1-score of 0.9999 for citeseer network with 8 dimensions in under 11 seconds
- For over 2000 real-world graphs, WEMBED consistently produces high-quality embeddings, especially for graphs with high heterogeneity
- WEMBED is competitive in running time with other state-of-the-art Euclidean embedders

## Why This Works (Mechanism)
WEMBED leverages the mathematical simplicity of Euclidean space while introducing weighted distances to capture heterogeneous graph properties. By assigning different weights to nodes based on their degrees (wv = deg(v)^(d/8)), the method can represent nodes with different connectivity patterns more accurately in low-dimensional space. The weighted distance metric allows the embedding to naturally separate nodes with different roles or importance levels without requiring complex non-Euclidean geometries.
