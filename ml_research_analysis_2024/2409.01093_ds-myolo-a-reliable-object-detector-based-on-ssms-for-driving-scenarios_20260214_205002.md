# DS MYOLO: A Reliable Object Detector Based on SSMs for Driving Scenarios

## Quick Facts
- arXiv ID: 2409.01093
- Source URL: https://arxiv.org/abs/2409.01093
- Reference count: 40
- DS MYOLO achieves state-of-the-art performance among similarly scaled YOLO series detectors, with the DS MYOLO-N model achieving 52.22% mAP on CCTSDB 2021 and 72.3% mAP on VLD-45.

## Executive Summary
DS MYOLO introduces a simplified selective scanning fusion block (SimVSS Block) based on State Space Models to enhance global feature fusion in real-time object detection for driving scenarios. The method also incorporates an efficient channel attention convolution (ECAConv) to improve cross-channel feature interactions while maintaining low computational complexity. Extensive experiments on CCTSDB 2021 and VLD-45 datasets demonstrate that DS MYOLO achieves state-of-the-art performance among similarly scaled YOLO series detectors, with the DS MYOLO-N model achieving 52.22% mAP on CCTSDB 2021 and 72.3% mAP on VLD-45. The detector shows strong competitiveness in precision, recall, and overall detection performance while maintaining real-time capabilities.

## Method Summary
DS MYOLO enhances traditional YOLO detectors by integrating a simplified selective scanning fusion block (SimVSS Block) based on State Space Models (SSMs) to improve global feature fusion. Additionally, an efficient channel attention convolution (ECAConv) is introduced to enhance cross-channel feature interactions with minimal computational overhead. These innovations are designed to address the challenges of object detection in driving scenarios, where precise and reliable detection is critical. The method is evaluated on CCTSDB 2021 and VLD-45 datasets, demonstrating significant improvements in mAP and overall detection performance compared to state-of-the-art models.

## Key Results
- DS MYOLO-N achieves 52.22% mAP on CCTSDB 2021, outperforming other YOLO series detectors of similar scale.
- DS MYOLO-N achieves 72.3% mAP on VLD-45, demonstrating strong performance in driving scenarios.
- The method maintains real-time capabilities while improving precision, recall, and overall detection performance.

## Why This Works (Mechanism)
The SimVSS Block leverages State Space Models (SSMs) to enable selective scanning and fusion of global features, addressing the limitations of traditional convolution-based feature fusion. This mechanism enhances the model's ability to capture long-range dependencies and contextual information, which is crucial for detecting objects in complex driving environments. The ECAConv module further improves feature representation by emphasizing important channels while reducing computational complexity. Together, these components enable DS MYOLO to achieve superior detection performance without compromising real-time efficiency.

## Limitations
