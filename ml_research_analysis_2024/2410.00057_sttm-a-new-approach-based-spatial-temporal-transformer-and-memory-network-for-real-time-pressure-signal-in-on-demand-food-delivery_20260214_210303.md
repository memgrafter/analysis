# STTM: A New Approach Based Spatial-Temporal Transformer And Memory Network For Real-time Pressure Signal In On-demand Food Delivery

## Quick Facts
- **arXiv ID:** 2410.00057
- **Source URL:** https://arxiv.org/abs/2410.00057
- **Reference count:** 26
- **Primary result:** Proposes STTM approach achieving 9.66% MAE improvement over competitors for RPS prediction in food delivery

## Executive Summary
This paper introduces STTM, a novel approach for predicting Real-time Pressure Signal (RPS) in On-demand Food Delivery (OFD) services. RPS measures the current pressure on the logistics system, enabling platforms to quickly respond to prevent overload. The method addresses limitations of previous models that inadequately utilize unique temporal and spatial characteristics of OFD services and lack sensitivity during abnormal events like severe weather or peak periods.

The core innovation combines a Spatio-Temporal Transformer structure with a Memory Network. The transformer learns logistics features across temporal and spatial dimensions by encoding historical information of business districts and their neighbors, while the Memory Network enhances sensitivity to abnormal events through attention mechanisms that highlight important features during anomalies. Experiments on Ele.me data demonstrate significant performance improvements, with the method successfully deployed online.

## Method Summary
STTM integrates a Spatio-Temporal Transformer with a Memory Network to predict Real-time Pressure Signal in on-demand food delivery. The transformer captures temporal dependencies and spatial correlations between business districts by processing historical RPS data, while the Memory Network improves anomaly detection through attention-based feature highlighting. This dual-component architecture addresses the unique challenges of OFD logistics, including rapid demand fluctuations and geographically distributed operations.

## Key Results
- STTM achieves 9.66% improvement in MAE compared to the best competitor method
- Successfully deployed online with demonstrated effectiveness in A/B testing
- Outperforms previous methods that inadequately capture temporal-spatial characteristics and anomaly sensitivity

## Why This Works (Mechanism)
The approach works by leveraging transformer-based attention mechanisms to capture complex temporal patterns and spatial relationships between business districts. The Memory Network component specifically addresses the challenge of detecting and responding to abnormal events by maintaining historical context and emphasizing relevant features during anomalies. This dual approach enables the model to handle both regular demand patterns and unexpected disruptions effectively.

## Architecture Onboarding

**Component Map:**
STTM -> Spatio-Temporal Transformer -> Memory Network -> RPS Prediction

**Critical Path:**
Input RPS data → Spatio-Temporal Transformer encoding → Memory Network attention → Final prediction output

**Design Tradeoffs:**
The approach trades increased model complexity for improved anomaly sensitivity and spatial-temporal modeling accuracy. The transformer component requires more computational resources but provides superior pattern recognition compared to traditional sequential models. The Memory Network adds overhead but significantly improves performance during abnormal events.

**Failure Signatures:**
Potential failures include: 1) Over-reliance on spatial correlations when local anomalies occur, 2) Memory Network saturation during prolonged abnormal periods, 3) Transformer attention collapse to dominant patterns, missing subtle variations.
