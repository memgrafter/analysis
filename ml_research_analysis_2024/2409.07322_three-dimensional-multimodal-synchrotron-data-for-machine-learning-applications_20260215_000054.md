# Three-Dimensional, Multimodal Synchrotron Data for Machine Learning Applications

## Quick Facts
- arXiv ID: 2409.07322
- Source URL: https://arxiv.org/abs/2409.07322
- Reference count: 40
- Primary result: A unique, multimodal synchrotron dataset of zinc-doped Zeolite 13X for developing machine learning techniques

## Executive Summary
This paper presents a comprehensive, multimodal synchrotron dataset designed to advance machine learning applications in materials science. The dataset includes four-resolution micro X-ray computed tomography (XCT) data and spatially resolved X-ray diffraction computed tomography (XRD-CT) data from the same zinc-doped Zeolite 13X sample. The data is provided in both raw and processed formats through Zenodo repositories, enabling development of super-resolution models, multimodal data fusion techniques, and 3D reconstruction algorithms.

## Method Summary
The dataset was acquired using synchrotron facilities at Diamond Light Source. Multi-resolution XCT imaging was performed on the I13-2 beamline at four pixel sizes (2.6µm, 1.625µm, 0.8125µm, and 0.325µm), while XRD-CT data was collected from the DIAD beamline at two spot sizes (25µm and 50µm). The sample was a zinc-doped Zeolite 13X pellet designed with spatially isolated zinc and sodium phases. Data is provided as both raw projections and reconstructed volumes, along with processing code and visualization notebooks.

## Key Results
- Multi-resolution XCT data provides four different spatial resolutions of the same sample region
- XRD-CT data offers chemical phase information with spatial resolution at two different spot sizes
- Both raw and processed data formats are available for algorithm development
- The dataset supports development of super-resolution, multimodal fusion, and 3D reconstruction techniques

## Architecture Onboarding

- Component map: XCT data (I13-2 beamline) -> XRD-CT data (DIAD beamline) -> Powder diffraction references -> Processing code and notebooks
- Critical path: To use this dataset for multimodal fusion, the critical path is: (1) load and register the multi-resolution XCT data, (2) load and process the XRD-CT data for sodium and zinc phases, (3) spatially align XCT and XRD-CT datasets using fiducial markers, (4) develop and train fusion models using the aligned data.
- Design tradeoffs: The dataset provides high-quality data but requires significant computational resources for processing (especially the 4000 projections per scan). The 0.325µm resolution XCT has a smaller field of view that doesn't capture the entire sample. The XRD-CT acquisition is time-intensive (up to 9 hours per scan).
- Failure signatures: Poor registration between modalities would manifest as misalignment artifacts in fusion results. Inconsistent color normalization across XCT resolutions could affect super-resolution training. Insufficient contrast between zinc and sodium phases would limit phase separation in XRD-CT.
- First 3 experiments:
  1. Load the 2.6µm and 0.325µm XCT datasets, perform basic visualization, and verify spatial registration by checking feature alignment.
  2. Load the XRD-CT data, perform azimuthal integration using provided code, and reconstruct sodium and zinc phase images to verify phase separation.
  3. Develop a simple multimodal fusion model that combines XCT structural features with XRD-CT chemical information for a small region of interest.

## Limitations

- The dataset is limited to a single sample type (zinc-doped Zeolite 13X), which may limit generalizability of trained models
- XRD-CT data has lower spatial resolution than XCT data, potentially limiting certain multimodal fusion approaches
- The quality of spatial registration between modalities is not explicitly validated in the paper
