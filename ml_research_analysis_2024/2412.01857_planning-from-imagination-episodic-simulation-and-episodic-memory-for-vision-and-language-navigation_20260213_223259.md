# Planning from Imagination: Episodic Simulation and Episodic Memory for Vision-and-Language Navigation

## Quick Facts
- arXiv ID: 2412.01857
- Source URL: https://arxiv.org/abs/2412.01857
- Authors: Yiyuan Pan; Yunzhe Xu; Zhe Liu; Hesheng Wang
- Reference count: 15
- Primary result: Achieves 8% SPL improvement on R2R and 4% improvement on REVERIE in unseen environments

## Executive Summary
This paper introduces SALI, a vision-and-language navigation agent that mimics human episodic simulation and memory mechanisms to improve navigation in unseen environments. SALI constructs a hybrid memory system that combines real observations with imagined future scenes, enabling long-term reasoning and decision-making. The method employs a recurrent imagination module that generates high-fidelity RGB images of potential future scenes, integrating these predictions into a topological map alongside real-world data. SALI achieves state-of-the-art performance on R2R and REVERIE benchmarks, demonstrating enhanced ability to process complex spatial relationships and object-room associations.

## Method Summary
SALI implements a hybrid reality-imagination memory system with a recurrent imagination module for VLN tasks. The method uses R2R and REVERIE datasets with pre-trained ViT/ResNet encoders and LXMERT for multimodal encoding. The system employs a topological memory map that stores both real observations and imagined RGB images, with a recurrent imagination tree that iteratively refines predictions using historical information. Training involves pre-training with masked language modeling, masked region classification, single-step action prediction, and object grounding, followed by fine-tuning with dynamic action fusion policy. The method achieves improved performance through its ability to reason about future states and make informed navigation decisions based on both real and imagined information.

## Key Results
- 8% improvement in Success Rate Weighted by Path Length (SPL) on R2R benchmark
- 4% improvement in Remote Grounding Success Rate Weighted by Path Length (RGSPL) on REVERIE benchmark
- Demonstrates enhanced performance in unseen environments with complex spatial relationships
- Shows particular strength in navigating environments with intricate instructions

## Architecture Onboarding

- Component map:
  Multimodal Transformer -> Recurrent Imagination Module -> Hybrid Memory System -> Action Fusion Policy

- Critical path:
  1. Update topological memory with current observation and historical information
  2. Generate imagined future scenes using recurrent imagination module
  3. Fuse imagined scenes into hybrid memory system
  4. Encode memory and instructions using multimodal transformer
  5. Generate navigation decisions based on fused scores from real and imagined nodes

- Design tradeoffs:
  - Memory vs. Computational Cost: Storing more imagined scenes improves reasoning but increases computational cost
  - Image Quality vs. Generation Speed: Higher quality imagined images provide better guidance but take longer to generate
  - Fusion Weighting: Balancing the influence of real vs. imagined information on navigation decisions

- Failure signatures:
  - Poor navigation performance in unseen environments
  - High computational cost during inference
  - Low quality of generated imagined scenes
  - Over-reliance on imagined information leading to poor real-world decisions

- First 3 experiments:
  1. Ablation study: Remove the imagination module and compare performance to full SALI
  2. Parameter sweep: Vary the fusion factor weight and analyze impact on navigation performance
  3. Qualitative analysis: Visualize generated imagined scenes and compare to real observations
