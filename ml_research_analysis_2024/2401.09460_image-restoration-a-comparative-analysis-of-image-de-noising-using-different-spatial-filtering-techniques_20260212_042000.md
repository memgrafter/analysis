# Image Restoration: A Comparative Analysis of Image De noising Using Different Spatial Filtering Techniques

## Quick Facts
- arXiv ID: 2401.09460
- Source URL: https://arxiv.org/abs/2401.09460
- Reference count: 18
- Primary result: Gaussian filter performs best on Gaussian, Poisson, and speckle noise; median filter excels at salt & pepper noise; bilateral filter preserves edges while denoising effectively.

## Executive Summary
This paper evaluates the effectiveness of four spatial filtering techniques—average, median, Gaussian, and bilateral filters—for removing different types of noise (Gaussian, Poisson, salt & pepper, and speckle) from grayscale images. The study systematically adds noise to clean images, applies each filter, and compares results using image quality metrics: PSNR, RMSE, MSE, SSIM, and UQI. The findings demonstrate that filter performance varies significantly depending on noise type, with the Gaussian filter showing superior performance on most noise types except salt & pepper, where the median filter excels. The bilateral filter is highlighted for its ability to preserve edges while denoising effectively.

## Method Summary
The study follows a systematic approach: clean grayscale images are corrupted with four types of noise (Gaussian, Poisson, Salt & Pepper, Speckle), then each of four spatial filters (Average, Median, Gaussian, Bilateral) is applied to the noisy images. Image quality is evaluated using five metrics: PSNR, RMSE, MSE, SSIM, and UQI. The process involves noise generation with unspecified parameters, filter application with unspecified kernel sizes and sigma values, and metric computation to compare filter performance across different noise types.

## Key Results
- Gaussian filter achieves highest PSNR for Gaussian, Poisson, and speckle noise types
- Median filter outperforms other filters specifically for salt & pepper noise removal
- Bilateral filter provides better edge preservation compared to Gaussian while maintaining denoising effectiveness
- Average filter demonstrates the least effective performance across all noise types and metrics

## Why This Works (Mechanism)
The effectiveness of different filters depends on how well their underlying mechanisms match the statistical properties of each noise type. Gaussian filters work well on Gaussian noise because they apply weighted averaging that matches the noise distribution's smooth characteristics. Median filters excel at salt & pepper noise removal because they directly replace extreme pixel values with median values from the neighborhood, effectively eliminating impulse noise. Bilateral filters preserve edges while denoising by considering both spatial proximity and intensity similarity, allowing them to smooth homogeneous regions while protecting edges from blurring. Average filters perform poorly because they indiscriminately average all neighboring pixels without distinguishing between noise and signal, causing significant detail loss across all noise types.

- Basis in paper: The paper mentions bilateral filter takes weighted sum based on spatial and intensity distances, but does not test different parameter values
- Why unresolved: The study used default or unspecified parameter values, so optimal settings for each noise type remain unknown
- What evidence would resolve it: Systematic comparison of bilateral filter performance (PSNR, SSIM, etc.) across a range of σs and σr values for each noise type

### Open Question 2
- Question: Would combining multiple filters (e.g., median followed by Gaussian) improve denoising performance compared to using a single filter for mixed noise scenarios?
- Basis in paper: The paper tests each filter independently but does not explore hybrid or sequential filtering approaches
- Why unresolved: Mixed or unknown noise types are common in practice, yet the paper only evaluates filters on single-noise models
- What evidence would resolve it: Experimental comparison of multi-stage filtering pipelines versus single-filter approaches on images corrupted with multiple noise types simultaneously

### Open Question 3
- Question: How do these spatial domain filters compare with frequency domain filters (e.g., Wiener filter) for the same noise types and evaluation metrics?
- Basis in paper: The paper explicitly states spatial domain was chosen for computational efficiency but does not compare with frequency domain methods
- Why unresolved: The paper only evaluates spatial domain filters, leaving questions about whether frequency domain methods might perform better or worse
- What evidence would resolve it: Direct performance comparison of spatial and frequency domain filters on identical noisy images using the same evaluation metrics (PSNR, SSIM, etc.)
