# SSTKG: Simple Spatio-Temporal Knowledge Graph for Intepretable and Versatile Dynamic Information Embedding

## Quick Facts
- arXiv ID: 2402.12132
- Source URL: https://arxiv.org/abs/2402.12132
- Authors: Ruiyi Yang; Flora D. Salim; Hao Xue
- Reference count: 40
- One-line primary result: SSTKG achieves 0.8016 acc@10 and 0.0452 RMS on Spend-Ohio dataset, outperforming baseline models in spatio-temporal prediction

## Executive Summary
This paper introduces SSTKG, a framework for interpretable and versatile dynamic information embedding using spatio-temporal knowledge graphs (STKGs). SSTKG addresses the challenge of incorporating dynamic spatio-temporal attributes into knowledge graphs to improve prediction and recommendation tasks. The framework uses a 3-step embedding approach that integrates spatial and temporal data through entity and relation embeddings. The paper demonstrates SSTKG's effectiveness on two datasets - Safegraph's Spend-Ohio retail data and TFNSW traffic data - showing superior performance over baseline models like SVR, LSTM, GRU, T-GCN, and ST-GCN in terms of accuracy metrics and RMS/RSD values.

## Method Summary
SSTKG employs a 3-step embedding approach that constructs a spatio-temporal knowledge graph by integrating spatial and temporal data into entities and relations. The framework first preprocesses datasets to extract entities, relationships, and facts, then constructs the SSTKG using distance thresholds. Entity embeddings and relations are trained using a loss function that accounts for both embedding quality and influence values. The model is evaluated on temporal sequence prediction tasks using accuracy metrics (acc@10, acc@15) and regression measures (RMS, RSD). The approach aims to provide interpretable results through its case study analysis while maintaining computational efficiency for dynamic information embedding.

## Key Results
- SSTKG achieves 0.8016 acc@10 and 0.0452 RMS on Spend-Ohio dataset
- SSTKG achieves 0.8095 acc@10 and 0.04245 RMS on TFNSW dataset
- Outperforms baseline models (SVR, LSTM, GRU, T-GCN, ST-GCN) in accuracy and regression metrics

## Why This Works (Mechanism)
SSTKG works by creating a unified framework that integrates spatial and temporal dimensions into knowledge graph embeddings. The key mechanism involves constructing a spatio-temporal knowledge graph where entities represent locations or objects with spatial coordinates, and relations capture temporal dynamics between these entities. By using a 3-step embedding process, SSTKG can effectively model the influence of both spatial proximity and temporal patterns on entity relationships. The framework's interpretability comes from its ability to trace predictions back to specific spatial-temporal relationships within the knowledge graph structure.
