# High-fidelity social learning via shared episodic memories enhances collaborative foraging through mnemonic convergence

## Quick Facts
- arXiv ID: 2412.20271
- Source URL: https://arxiv.org/abs/2412.20271
- Reference count: 40
- High-fidelity social learning consistently enhanced resource collection efficiency and distribution across memory lengths in collaborative foraging tasks.

## Executive Summary
This study investigates how the fidelity of social learning through shared episodic memories impacts collective foraging performance. Using Sequential Episodic Control (SEC) agents in a grid-world environment, the research systematically varies memory length, social learning frequency, and transfer noise to examine how accurate memory transmission affects group problem-solving. The findings reveal that high-fidelity social learning (low transfer noise) consistently improves resource collection and fosters mnemonic alignment among agents, while low-fidelity learning increases memory diversity without performance gains. The research also identifies an optimal range for episodic memory length, beyond which performance plateaus.

## Method Summary
The study employs SEC agents in a 2D grid-world foraging environment where agents collect fruits and deposit them in a nest. Agents have partial observability (3x3 field of view) and can perform five actions. Each agent maintains short-term memory (STM) and long-term memory (LTM) buffers, with the ability to share memories with visible neighbors based on transfer rate and transfer noise parameters. The experiments systematically vary STM length (10, 20, 30), transfer rate (0, 1, 10, 50), and transfer noise (0, 0.1) across 5000 episodes. Mnemonic metrics including relative diversity, group diversity, group alignment, and memory distribution are used to analyze collective memory structure and its relationship to performance.

## Key Results
- High-fidelity social learning (transfer noise = 0) consistently improved resource collection efficiency across all memory lengths tested
- Low-fidelity social learning (transfer noise = 0.1) increased mnemonic diversity but failed to translate into performance gains
- An optimal memory length range was identified (10-20 timesteps) beyond which performance plateaued
- Mnemonic alignment was significantly higher under high-fidelity conditions, correlating with improved resource distribution

## Why This Works (Mechanism)

### Mechanism 1
- Claim: High-fidelity social learning enhances collective foraging by enabling accurate propagation of successful behavioral sequences through shared episodic memories.
- Mechanism: Agents share complete episodic memory sequences via prioritized experience sharing, with higher reward sequences being more likely to be transmitted. This ensures that high-value strategies spread across the group.
- Core assumption: The fidelity of memory transmission (transfer noise) directly impacts the accuracy of the information being shared, with low noise enabling accurate copying of successful strategies.
- Evidence anchors:
  - [abstract] "High-fidelity social learning is shown to consistently enhance resource collection efficiency and distribution, with benefits sustained across memory lengths."
  - [section] "In other words, there is a prioritization in the retrieval of sequential memories proportional to the reward value associated with episodic memory."
  - [corpus] Weak - neighboring papers focus on memory architectures but not specifically on fidelity in social learning contexts.
- Break condition: When transfer noise exceeds a threshold where noise corrupts the reward signal, making it impossible to distinguish successful from unsuccessful strategies.

### Mechanism 2
- Claim: Mnemonic alignment between agents is a prerequisite for effective collective problem-solving in foraging tasks.
- Mechanism: High-fidelity social learning leads to convergence of episodic memories across agents, creating alignment that enables coordinated behavior and efficient resource exploitation.
- Core assumption: Aligned memories mean agents have similar representations of successful strategies, enabling them to coordinate implicitly without explicit communication.
- Evidence anchors:
  - [abstract] "Novel analyses using mnemonic metrics reveal that high-fidelity social learning also fosters mnemonic group alignment and equitable resource distribution"
  - [section] "More frequent high-fidelity social learning is beneficial for agents regardless of their memory capacity."
  - [corpus] Weak - neighboring papers discuss memory alignment but not in the context of social learning fidelity.
- Break condition: When memory diversity exceeds a threshold where agents no longer share overlapping successful strategies, preventing coordination.

### Mechanism 3
- Claim: There exists an optimal range for episodic memory length that balances complexity of stored behaviors with transmission fidelity.
- Mechanism: Memory length affects both the capacity to store complex behavioral sequences and the probability of transmission errors. Too short limits behavioral complexity; too long increases transmission noise.
- Core assumption: The relationship between memory length and task performance follows an inverted-U curve, with performance plateauing or declining beyond optimal length.
- Evidence anchors:
  - [abstract] "Additionally, we identify an optimal range for episodic memory length in this task, beyond which performance plateaus."
  - [section] "Interestingly, agents with larger short-term memory capacity (STM = 30) did not perform better than those with an average capacity (STM = 20)."
  - [corpus] Weak - neighboring papers discuss memory capacity but not specifically in the context of transmission fidelity.
- Break condition: When memory length becomes so long that transmission noise overwhelms the benefit of storing complex sequences.

## Open Questions the Paper Calls Out

### Open Question 1
- Question: How does the fidelity of social learning interact with memory length to influence collective exploration vs. exploitation dynamics in different foraging environments?
- Basis in paper: [inferred] The paper discusses how low-fidelity social learning increases mnemonic diversity and potentially promotes exploration, while high-fidelity learning enhances alignment and exploitation. However, it does not fully explore how these effects interact with varying memory lengths across different environmental conditions.
- Why unresolved: The study focuses on a single foraging environment and does not systematically vary environmental complexity or resource distribution to test how fidelity-memory interactions affect exploration-exploitation trade-offs.
- What evidence would resolve it: Experiments testing SEC agents with different memory lengths and fidelity levels across multiple environments (e.g., sparse vs. clustered resources, static vs. dynamic conditions) would clarify how these factors jointly influence collective foraging strategies.

### Open Question 2
- Question: What are the precise mechanisms by which low-fidelity social learning increases mnemonic diversity without translating into performance gains in collaborative foraging?
- Basis in paper: [explicit] The paper explicitly states that low-fidelity social learning increases mnemonic diversity but does not improve performance, contrasting with high-fidelity conditions. However, the underlying mechanisms remain unclear.
- Why unresolved: The study identifies the phenomenon but does not investigate the specific processes (e.g., noise propagation, memory interference) that lead to increased diversity without performance benefits.
- What evidence would resolve it: Detailed analysis of how noisy transmissions affect individual and group-level memory structures, combined with performance tracking, would reveal whether diversity arises from redundant or conflicting memories.

### Open Question 3
- Question: How does the balance between individual and social learning change with varying levels of social learning fidelity and frequency in multi-agent systems?
- Basis in paper: [inferred] The paper focuses on social learning dynamics but does not explore how agents might balance individual exploration with social learning, especially under different fidelity and frequency conditions.
- Why unresolved: The study abstracts away individual learning mechanisms, leaving the interaction between individual and social learning unexplored.
- What evidence would resolve it: Incorporating individual learning components into the SEC model and testing their interplay with social learning under varying fidelity and frequency conditions would clarify the optimal balance for collective performance.

## Limitations

- The study uses a simplified grid-world environment that may not capture the complexity of real-world collaborative tasks
- The fixed number of agents (4) and resources (4 fruits) creates a constrained scenario where social learning benefits might be amplified
- The transfer noise mechanism uses a simplified binary model (0 or 0.1) that may not reflect continuous fidelity loss in natural social learning

## Confidence

- **High Confidence**: The core finding that high-fidelity social learning consistently improves resource collection efficiency across memory lengths is well-supported by experimental results with clear statistical patterns.
- **Medium Confidence**: The optimal memory length range (10-20) showing performance plateaus is based on limited sampling points and may shift with different environmental configurations or task complexities.
- **Medium Confidence**: The relationship between mnemonic alignment and performance gains is inferred through correlational analysis rather than causal manipulation of alignment independently from fidelity.

## Next Checks

1. Test the robustness of the optimal memory length finding by running additional experiments with STM values at 15 and 25 to better characterize the performance plateau.
2. Conduct ablation studies isolating memory sharing from other social mechanisms by testing individual agent performance when given access to a "global memory" versus true social learning.
3. Implement a more granular transfer noise scale (0.01 increments from 0 to 0.1) to identify the precise threshold where social learning benefits break down and better understand the noise-fidelity relationship.
